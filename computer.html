Program Summary

This program includes Microsoft Word for typing and formating documents, Excel for working with data and performing calculations, PowerPoint for creating presentations and Access for Database Management.

Requirements

1. Basic English reading ability

2. A clean record of behaviour

3. Commitment to practice

4. Regular attendance

Learn basis Microsoft Office
  • Lesson 1.1
  • Computers & Microsoft Office
  • 02 days

In this lesson, one will be introduced to different computer parts, Microsoft Office Programs , opening and closing office applications, saving, managing and opening files and folders.

  • Lesson 1.2
  • Microsoft Word
  • 03 days

Here one will work with images, creating tables, page layout and simple templates.

  • Lesson 1.3
  • Microsoft Excel - Basics
  • 02 days

You will understand rows, columns and cells. Enter data in cells, format cells, use simple formulas and create basic tables.

  • Lesson 1.4
  • Excel Chats & Data Handling
  • 03 days

Create charts, sort & filter data.

  • Lesson 1.5
  • Microsoft PowerPoint
  • 03 days

Create a new presentation, add slides, choose layouts, insert texts, images and shapes, apply slide transitions and animations, prepare and present a slideshow.

  • Lesson 1.6
  • Introduction to Access
  • 03 days

Definition of a database, create tables in access, fields and data types, entering and viewing records, simple queries and forms.
